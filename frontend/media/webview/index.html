Backend not running

The BTL backend server is not reachable. This can happen if the server isn't started, or if the BTL_API_KEY is missing from backend/.env.

1 Make sure backend/.env contains your API key:
BTL_API_KEY=your-key-here
2 Start the backend server:
make run or docker compose up -d

The backend runs on localhost:52104 by default. Change it in Settings > BTL: API URL.

Invalid Gemini API key

The Gemini API rejected the request. Your API key may be invalid or expired.

1 Get a free key at aistudio.google.com/apikey
2 Update backend/.env:
BTL_API_KEY=your-new-key-here
3 Restart the backend:
make stop && make run
